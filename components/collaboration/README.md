# components/collaboration/ — Extended Collaboration UI

> Prestige Build — Prestige Technologie Company, fondée par Glody Dimputu Ngola.

Composants React pour les fonctionnalités de collaboration **au niveau du projet** (présence, commentaires, activité, partage).

| Composant | Description |
|-----------|-------------|
| `PresenceBar.tsx` | Barre affichant les utilisateurs en ligne dans un projet. |
| `UserCursor.tsx` | Curseur SVG d'un collaborateur distant sur une page. |
| `CommentThread.tsx` | Fil de commentaires avec ajout et résolution. |
| `ActivityFeed.tsx` | Flux d'activité récente du projet. |
| `ShareDialog.tsx` | Modale de partage et d'invitation par e-mail. |

Voir `lib/collaboration/README.md` pour la logique métier associée.
